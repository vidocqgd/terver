Задача 3. Проведите тест гипотезы. Продавец утверждает, что средний вес пачки печенья
составляет 200 г.
Из партии извлечена выборка из 10 пачек. Вес каждой пачки составляет:
202, 203, 199, 197, 195, 201, 200, 204, 194, 190.
Известно, что их веса распределены нормально.
Верно ли утверждение продавца, если учитывать, что уровень значимости 1%? (Провести двусторонний тест.)


Найдем выборочное среднее и выборочное стандартное отклонение:
Среднее значение выборки (x̄) = (202 + 203 + 199 + 197 + 195 + 201 + 200 + 204 + 194 + 190) / 10 = 198.5 г
Выборочное стандартное отклонение (s) = √((Σ(xi - x̄)^2) / (n - 1)) = √((∑(xi^2) - (n * x̄^2)) / (n - 1)) ≈ 4.45 г

Вычисление статистики теста:
t-статистика = (x̄ - μ) / (s / √n), где μ - заданное значение (200 г), n - размер выборки

t-статистика = (198.5 - 200) / (4.45 / √10) = -1.065

Определение критической области и принятие решения:

Критическая область: двухсторонняя область с α/2 = 0.01/2 = 0.005 с каждой стороны.
Критические значения можно найти с помощью таблицы распределения Стьюдента Для выборки размером 10 и α = 0.01, критические значения t составляют  ±3.169.
Так как |t-статистика| = |-1.065| < 3.169, мы не отвергаем нулевую гипотезу.
